
Консольное Java приложение, которое симулирует базовую систему управления жителями и их картами доступа.


Функциональность:

1. Добавление контактов:
   - Приложение позволяет пользователю добавлять новые контакты.
   - Для каждого контакта имеется возможность сохранять следующую информацию:
     - Имя
     - Фамилия
     - Номер пропуска (Уникальное поле, String, длина поля 8 символов, разрешенные символы: 0-9A-F. Пример строки: "00F1C313")

2. Просмотр контактов:
   - Отображает список всех добавленных контактов с их информацией.
   - Отображает только номера пропусков

3. Поиск контактов:
   - Реализована возможность поиска контактов по номеру пропуска.

4. Удаление контактов:
   - Предоставлена возможность удаления контакта по его уникальному идентификатору.

5. Обработка ошибок:
   - Предусмотрена обработка возможных исключений и некорректного ввода данных Уникального Номер пропуска.
   - Приложение выводит в консоль причину исключения (красный текст) и не завершаться с ошибкой при неверном вводе данных.

6. Сохранение данных:
  - Реализована возможность сохранения контактов в файл.
  - Реализована возможность загрузки ранее сохранённых контактов из файла (чтение данных из файла).

-------------------------------------------------------------------------------------------------------------

        Как работать с приложением


1. Работаем только с классом Main

1.1 Добавление контактов
        - для удобства, в классе продублировано частичное описание функционала программы
        - все функциональное описание закомментировано между символами /* ... */ , удалять или снимать символы комментария не следует
        - Символами "//" временно закомментирована часть рабочего кода
        - снимаем ограничения (удаляем символы //) по мере необходимости работы с нужным участком кода

1.2 Включаем генератор уникального номера пропуска
        - снимаем ограничение "//" и запускаем Генератор уникального номера пропуска для нового Контакта

1.3 Создаём новые Контакты
        - в Конструкторе добавляем новый контакт и вставляем ранее сгенерированный номера пропуска
        - при попытке ввести некорректный Номер пропуска в консоль выведется предупреждение об ошибке (Exception)

1.4  Добавляем Контакты в список


2.   Просмотр контактов с их информацией
        - снимаем ограничения (удаляем символы //) по мере необходимости работы с нужным участком кода

2.1   Просмотр только номеров пропусков
        - снимаем ограничения (удаляем символы //) по мере необходимости работы с нужным участком кода

3.   Поиск контактов по номеру пропуска
        - снимаем ограничения (удаляем символы //) по мере необходимости работы с нужным участком кода

4.   Удаление контактов по ключу
        - снимаем ограничения (удаляем символы //) по мере необходимости работы с нужным участком кода


6.   Сохранение данных контактов в файл и (чтение сохранённых данных из файла)
        - снимаем ограничения (удаляем символы //) по мере необходимости работы с нужным участком кода
